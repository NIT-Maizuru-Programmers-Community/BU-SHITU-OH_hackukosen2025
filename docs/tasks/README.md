## 優先度について

アプリ仕様書で定められた優先度:

1. 個人のポイント数閲覧 → タスク03
2. IC の登録動作(NFC) → タスク02
3. 本日のレース結果 → タスク05
4. リーダーボード(ランキング) → タスク04
5. 自分のポイント履歴 → タスク03
6. 部室にいる人がリアルタイムでわかる → タスク07
