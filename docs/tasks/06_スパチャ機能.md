## Firestore スキーマ

```typescript
// transfers コレクション
interface Transfer {
  id: string;
  senderId: string;
  senderName: string;
  receiverId: string;
  receiverName: string;
  amount: number;
  message?: string;
  createdAt: Timestamp;
}
```

## 画面一覧

| 画面パス             | 説明             |
| -------------------- | ---------------- |
| `/transfer`          | 送金画面         |
| `/transfer/history`  | 送金履歴         |

## コンポーネント

- `UserSelector`: 送金先ユーザー選択
- `TransferForm`: 送金フォーム
- `TransferConfirmDialog`: 送金確認ダイアログ
- `TransferNotification`: 受信通知
- `TransferHistoryList`: 送金履歴一覧
- `TransferHistoryItem`: 送金履歴アイテム
