# タスク05: レース機能

## 概要

毎日開催されるレース（ベット機能）を実装する。部員の部室滞在時間を競い、ベットにより盛り上げる

## Firestore スキーマ

```typescript
// races コレクション
interface Race {
  id: string;
  date: string; // "2025-12-10"
  status: 'open' | 'closed' | 'finished';
  participants: string[]; // ユーザーUID配列
  results?: RaceResult[];
  startTime: Timestamp;
  endTime: Timestamp;
  createdAt: Timestamp;
}

interface RaceResult {
  oderId: string;
  rank: number;
  stayDuration: number; // 滞在時間（分）
}

// bets コレクション
interface Bet {
  id: string;
  raceId: string;
  oderId: string; // ベットした人
  targetUserId: string; // ベット対象
  amount: number;
  result?: 'win' | 'lose';
  payout?: number;
  createdAt: Timestamp;
}
```

- `BetConfirmDialog`: ベット確認ダイアログ
- `RaceResult`: 結果表示
- `RaceResultAnimation`: 結果発表アニメーション
- `RaceHistoryList`: 過去のレース一覧

## レースルール

1. 毎日自動でレースが開催される
2. 部室にログインした部員が自動的に参加者となる
3. ログイン時に他の部員にベットできる（1pt〜）
4. 終了時刻（設定可能）にレース終了
5. 滞在時間が長い順に順位が決まる
6. ベット結果に応じてポイントが変動

## 配当計算例

- 1位にベット → 2倍
- 2位にベット → 1.5倍
- 3位にベット → 1.2倍
- 4位以下 → 没収
