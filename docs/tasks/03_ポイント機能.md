# タスク03: ポイント機能

## 概要

ユーザーのポイント閲覧・履歴表示機能を実装する

## 優先度

- ポイント閲覧: **1**（最優先）
- ポイント履歴: **5**

## 要件 ID

- F-001: ポイント付与
- F-002: ポイント閲覧
- F-003: ポイント履歴
- F-004: ポイントリセット

## 前提条件

- タスク02（認証機能）が完了していること

## タスク一覧

### 1. ポイント閲覧機能（優先度1）

- [ ] ホーム画面にポイント表示エリア作成
- [ ] 現在のポイント数をリアルタイム表示
- [ ] ポイント数の視覚的な演出（アニメーション等）
- [ ] ポイント取得時の通知表示

### 2. ポイント履歴機能（優先度5）

- [ ] `/history` ページ作成
- [ ] 履歴一覧表示コンポーネント
- [ ] フィルタリング機能（取得/使用/送金）
- [ ] 日付でのグルーピング表示
- [ ] ページネーション or 無限スクロール

### 3. ポイント管理ロジック

- [ ] ポイント付与関数 (`lib/points.ts`)
- [ ] ポイント減算関数
- [ ] ポイント履歴記録関数
- [ ] トランザクション処理の実装

### 4. ポイントリセット機能

- [ ] リセットスケジュール設定（Cloud Functions）
- [ ] リセット前の通知機能
- [ ] リセット履歴の保存

## Firestore スキーマ

```typescript
// points コレクション（履歴）
interface PointHistory {
  id: string;
  oderId: string;
  amount: number; // 正: 獲得, 負: 使用
  type: 'login_bonus' | 'race_win' | 'race_bet' | 'transfer_send' | 'transfer_receive' | 'reset';
  description: string;
  relatedId?: string; // レースID や 送金相手のUID
  createdAt: Timestamp;
}
```

## 画面一覧

| 画面パス   | 説明                     |
| ---------- | ------------------------ |
| `/`        | ホーム（ポイント表示）   |
| `/history` | ポイント履歴一覧         |

## コンポーネント

- `PointDisplay`: ポイント数表示
- `PointHistoryList`: 履歴一覧
- `PointHistoryItem`: 履歴アイテム
- `PointBadge`: ポイント変動バッジ

## 成果物

- ホーム画面でのポイント表示
- ポイント履歴ページ
- ポイント管理ユーティリティ
