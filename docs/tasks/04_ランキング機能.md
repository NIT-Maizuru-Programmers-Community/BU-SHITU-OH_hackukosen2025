## Firestore クエリ

```typescript
// ランキング取得
const getRanking = async (limit: number = 50) => {
  const usersRef = collection(db, 'users');
  const q = query(usersRef, orderBy('totalPoints', 'desc'), limit(limit));
  return getDocs(q);
};
```

## 画面一覧

| 画面パス   | 説明             |
| ---------- | ---------------- |
| `/ranking` | ランキング一覧   |

## コンポーネント

- `RankingList`: ランキング一覧
- `RankingItem`: ランキングアイテム
- `TopThree`: トップ3表示（特別デザイン）
- `CrownBadge`: 社長席バッジ
- `MyRankCard`: 自分の順位カード
