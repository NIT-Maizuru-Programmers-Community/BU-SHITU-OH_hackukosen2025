# タスク01: UI 構築

## 概要

モバイルファーストの共通 UI コンポーネントとレイアウトを構築する

## 前提条件

- タスク00（初期セットアップ）が完了していること
- shadcn/ui がインストール済みであること

## タスク一覧

### 1. カラーテーマ・デザイントークン

- [ ] カラーパレットの定義 (`globals.css`)
  - プライマリカラー
  - セカンダリカラー
  - アクセントカラー
  - 背景色・テキスト色
- [ ] フォント設定
- [ ] スペーシング・サイズの統一

### 2. 共通レイアウト

- [ ] `components/layout/AppLayout.tsx` 作成
- [ ] ヘッダーコンポーネント
  - ロゴ・タイトル
  - 通知アイコン
  - ユーザーアバター
- [ ] ボトムナビゲーション
  - ホーム
  - ランキング
  - レース
  - 履歴
  - 設定

### 3. 共通コンポーネント

- [ ] `components/ui/PointBadge.tsx`: ポイント表示バッジ
- [ ] `components/ui/UserAvatar.tsx`: ユーザーアバター
- [ ] `components/ui/RankBadge.tsx`: 順位バッジ（1〜3位は特別デザイン）
- [ ] `components/ui/StatusBadge.tsx`: ステータス表示
- [ ] `components/ui/EmptyState.tsx`: 空状態表示
- [ ] `components/ui/LoadingSpinner.tsx`: ローディング表示

### 4. ページテンプレート

- [ ] `components/layout/PageHeader.tsx`: ページヘッダー
- [ ] `components/layout/PageContainer.tsx`: ページコンテナ
- [ ] `components/layout/Card.tsx`: カードコンポーネント
- [ ] `components/layout/Section.tsx`: セクションコンポーネント

### 5. ホーム画面 UI

- [ ] `/` ページのレイアウト作成
- [ ] ポイント表示エリア
- [ ] 在室者サマリーエリア
- [ ] ランキングサマリーエリア
- [ ] 本日のレースエリア

### 6. アニメーション・インタラクション

- [ ] ページ遷移アニメーション
- [ ] ポイント増減アニメーション
- [ ] プルトゥリフレッシュ
- [ ] タップフィードバック

## ディレクトリ構成

```
app/
├── components/
│   ├── layout/
│   │   ├── AppLayout.tsx
│   │   ├── Header.tsx
│   │   ├── BottomNav.tsx
│   │   ├── PageHeader.tsx
│   │   └── PageContainer.tsx
│   └── ui/
│       ├── PointBadge.tsx
│       ├── UserAvatar.tsx
│       ├── RankBadge.tsx
│       ├── StatusBadge.tsx
│       ├── EmptyState.tsx
│       └── LoadingSpinner.tsx
├── globals.css
└── layout.tsx
```

## デザイン方針

### モバイルファースト

- 基本幅: 375px（iPhone SE）〜 428px（iPhone 14 Pro Max）
- タッチターゲット: 最小 44x44px
- フォントサイズ: 最小 14px

### ボトムナビゲーション

```
┌─────────────────────────────────────┐
│                                     │
│           メインコンテンツ            │
│                                     │
├─────────────────────────────────────┤
│  🏠    📊    🏁    📜    ⚙️  │
│ ホーム ランキング レース 履歴 設定    │
└─────────────────────────────────────┘
```

### カラーテーマ案

```css
:root {
  /* Primary - ゴールド系（王様イメージ） */
  --primary: #F59E0B;
  --primary-foreground: #FFFFFF;
  
  /* Secondary */
  --secondary: #3B82F6;
  --secondary-foreground: #FFFFFF;
  
  /* Background */
  --background: #0F172A;
  --foreground: #F8FAFC;
  
  /* Card */
  --card: #1E293B;
  --card-foreground: #F8FAFC;
  
  /* Accent */
  --accent: #8B5CF6;
  
  /* Ranking Colors */
  --gold: #FFD700;
  --silver: #C0C0C0;
  --bronze: #CD7F32;
}
```

- 統一されたデザインシステム
- 共通レイアウト（ヘッダー、ボトムナビ）
- 再利用可能な UI コンポーネント
- ホーム画面の基本 UI
