# タスク02: 認証機能

## 概要

Firebase Authentication を使用したユーザー認証と NFC カード登録機能を実装する

## 優先度

- NFC 登録: **2**（アプリ仕様より）

## 要件 ID

- F-013: NFC 登録
- F-014: NFC ログイン
- F-015: ログインボーナス

## 前提条件

- タスク00（初期セットアップ）が完了していること
- タスク01（UI 構築）が完了していること

## タスク一覧

### 1. Firebase Authentication 設定

- [ ] Firebase Console で Authentication を有効化
- [ ] 認証方法の設定（メール/パスワード or 匿名認証）
- [ ] 認証コンテキストの作成 (`contexts/AuthContext.tsx`)
- [ ] 認証フックの作成 (`hooks/useAuth.ts`)

### 2. ユーザー登録画面

- [ ] `/register` ページ作成
- [ ] ユーザー名入力フォーム
- [ ] 登録処理の実装
- [ ] Firestore への初期ユーザーデータ保存

### 3. ログイン画面

- [ ] `/login` ページ作成
- [ ] ログインフォーム
- [ ] ログイン処理の実装

### 4. NFC カード登録機能

- [ ] `/settings/nfc` ページ作成
- [ ] NFC カード ID 入力/読み取り UI
- [ ] カード ID と ユーザーの紐付け処理
- [ ] 登録済みカードの確認・削除機能

### 5. ログインボーナス処理

- [ ] ログイン時のボーナスポイント付与ロジック
- [ ] 1 日 1 回制限の実装
- [ ] ボーナス付与通知 UI

### 6. 認証ガード

- [ ] 未認証ユーザーのリダイレクト
- [ ] 保護されたルートの設定

## Firestore スキーマ

```typescript
// users コレクション
interface User {
  uid: string;
  displayName: string;
  nfcCardId?: string;
  totalPoints: number;
  lastLoginDate: Timestamp;
  lastBonusDate?: Timestamp;
  createdAt: Timestamp;
  updatedAt: Timestamp;
}
```
